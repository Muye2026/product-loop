# Desktop Device Demo Brief

## Goal

Create a compact monitor-top desk companion that helps a remote worker notice posture drift and presence state without feeling like a toy or an enterprise security camera.

## Audience

Primary audience: remote knowledge workers using 24-27 inch desktop monitors in shared or home office settings.

## Scenario

The device lives on top of a desktop monitor all day. It should remain low-profile during work, provide quick status cues at a glance, and allow simple local interaction without requiring a phone.

## Primary functions

- detect user presence or absence
- provide posture or wellness cues
- show short status messages or icons
- support one-button local acknowledgement
- emit light audio prompts when needed

## Modules and assumptions

- compact front display module
- camera or vision sensor package
- main logic board with service port access
- small speaker or buzzer
- one physical button

## Hard constraints

- fit common 24-27 inch desktop monitors
- remain visually quiet from the front
- avoid spring-clamp complexity
- preserve basic service access without full disassembly
- keep total envelope close to small desktop-hardware scale

## Soft preferences

- look like a refined tool, not a toy
- feel friendly but not character-like
- maintain left-to-right front-face order
- keep mounting logic believable from the side view

## Unknowns

- exact board package
- final display module size
- exact camera field-of-view and tilt
- preferred material split between metal-like and plastic-like surfaces
